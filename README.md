__我用了两种方式实现了粒子动态连线效果。__

心想总不会两两比较吧，这样频繁的重绘计算浏览器能吃的消吗，看了一个particles大项目的源码，发现还真是，是我低估了浏览器的计算能力...）__    

__用canvas实现明显浏览器会更从容，我还添加了鼠标move连线效果，svg还是想想算了（应该是我的方式错了，期待后续找到更优解法）__
